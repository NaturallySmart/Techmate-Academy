Array Methods

Merging (Concatenating) Arrays

The concat() method creates a new array by merging (concatenating) existing arrays:

It does not change the existing arrays. It always returns a new array and can take any number of arguments to merge into one array. It can also take strings as arguments

See code sample below: 

                const boysNames = ['Mike', 'John', 'Frank'];
                const girlsNames = ['Joy', 'Bella', 'Kate', 'Anna']
                let allNames = boysNames.concat(girlsNames); // Join two arrays

Splicing and Slicing Arrays

.splice() Method

The splice() method adds new items to an array. It can also be used to delete an element from an specific index in an array. It usually takes arguments into it's parenthesis such as .splice(2,1) which means items should be modified starting from index 2 and also delete 1 element starting from that index.

See code sample below: 

                const fruits = ["apple", "banana", "cherry"];
                fruits.splice(1, 0, "orange"); 
                
                // Add orange at index 1 and deletes no element
Adding two elements and deleting 2 element from the array starting from index 1. 
                const names = ['John', 'Kate', 'Mike', 'Joy'];
                names.splice(1, 2, 'Julia', 'Ray')
    
                //This will remove Kate and Mike and add Julia and Ray to that 
                position respectively.
