Parameters and Arguments

In the previous lesson, we were able to declare and call a parameter without using a parameter. However, when coding most times, there is need to pass a parameter to the function. You can see this as a customisation of your function in the example in the last lesson.

In JavaScript, parameters are variables that you define inside the parentheses of a function declaration or expression. They represent values that will be passed into the function when called. A parameter can be seen as placeholders for the information to be passed when the fuction is called. You can pass one or more parameter(s) into a function

Synthax:

function myFunction(parameter1, parameter2, parameter3) {
code here;
}
