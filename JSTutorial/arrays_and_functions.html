Arrays and Functions

We can pass an array into a function. When an array has been mutated or changed, using any of the method talked about in the last lesson inside a function, such changes are reflected inside the function as well.
Here's an example 

                    const colors = ['Red', 'Blue', 'Yellow'];
    
                    function addColor(arr) {
                        arr.push('Green');
                    }
    
                    addColor(colors);
    
                    console.log(colors); // Outputs['Red', 'Blue', 'Yellow', 'Green'];
Here is what happened in the above example:
  • The color array has 3 elements.
  • The function addColor() has a parameter of arr and uses .push() to add a color 'Green' element into arr.
  • We call addColor() with an argument of colors which will execute the code inside addColor().
  • We check the value of colors and it now includes the 'Green' element! The array was mutated!
