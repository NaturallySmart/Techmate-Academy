What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

<tagname>content</tagname>

For example, the paragraph above was created with this code: 

            <p>
                An HTML element is defined by a start tag,
                some content, and an end tag.
            </p>
HTML contains many elements. While some elements has content, others are empty. For instance the p tag has n opening and closing tag and contains element within it. same things as the h tags. Other tags such as the br tag, b tag etc are empty tags as they have no elements within them.

HTML Tags

Here is a list of some HTML Tags

  • h1 to h6 tags: This defines an HTML heading
  • p tag: This defines an HTML paragraph
  • img tag: This displays an image on your webpage
  • a tag: This creates hyperlinks in webpages
  • div tag: It divides the page into different sections
  • span tag: It groups inline elements together
  • ul tag: Unordered List
  • ol tag: Ordered List
  • li tag: List item
  • table tag: Table structure
  • tr tag: Rows in table
  • td tag: Data cells in table
  • th tag: Header cells in table
  • form tag: Form for user input
  • input tag: Input field for user data
  • button tag: Button that can be clicked
  • select tag: Dropdown menu for options
  • option tag: Option in dropdown menu
  • textarea tag: Text area for multi line text input
  • label tag: Associates a caption with a specific form control
  • em tag: Emphasizes important words or phrases
  • strong tag: Important text
  • i tag: italicized text
  • u tag: Underlined text
  • b tag: to make text bold
  • br tag: breaks content to a new line
