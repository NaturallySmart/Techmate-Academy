What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

<tagname>content</tagname>

For example, the paragraph above was created with this code: 

            <p>
                An HTML element is defined by a start tag,
                some content, and an end tag.
            </p>
HTML contains many elements. While some elements has content, others are empty. For instance the p tag has n opening and closing tag and contains element within it. same things as the h tags. Other tags such as the br tag, b tag etc are empty tags as they have no elements within them.
