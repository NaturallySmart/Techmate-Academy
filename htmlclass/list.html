HTML List

HTML lists are used to group related items together on a webpage.

There are two types of list in HTML: ordered and unordered lists.

Unordered List

An unordered list is used to list items in no particular ranking or order as the name implies, Unordered.

An unordered list is created using the <ul> tag. Each item within an unordered list is represented by the <li> tag.

See example below: 

            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ul>

Output:

  • Item 1
  • Item 2
  • Item 3

Now let's make a list of four car makers in no particular order: 

            <ul>
                <li>Toyota</li>
                <li>Honda</li>
                <li>Ford</li>
                <li>Chevrolet</li>
            </ul>

Output:

  • Toyota
  • Honda
  • Ford
  • Chevrolet

Ordered List

An ordered list is used to list items in a specific and logical sequence as the name implies, Ordered.

An ordered list is created using the <ol> tag. Each item within an ordered list is represented by the <li> tag. The value inside the opening <li> tag represents the number for that item. If you don't provide any values it will start from 1 automatically. You can also manually set the starting point with the attribute start="n", where n is your desired starting number. Here are some examples: 

            <ol>
                <li>First Item</li>
                <li>Second Item</li>
                <li>Third Item</li>
            </ol>

Output:

  1. First Item
  2. Second Item
  3. Third Item
            <ol start="5">
                <li>Fifth Item</li>
                <li>Sixth Item</li>
                <li>Seventh Item</li>
            </ol>

Output:

  1. Fifth Item
  2. Sixth Item
  3. Seventh Item

Note: Ordered lists should be used when listing things in a specific order such as recipes or instructions on how to do something. Unordered lists can be used when listing random information like blog posts or comments. List can also be used when creating a navigation bar and footer.
