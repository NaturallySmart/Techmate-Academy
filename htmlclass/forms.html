HTML Forms

Forms are tools used to collect information or data from people. Forms are a very important part of our everyday life. Students fill forms in school, customers fill forms in banks and other businesses etc.

When creating a website or application, there are high chances that we will need forms to either collect a user's details, for a user to log in or to receive message or feedback from users etc. Examples of forms we use when creating a website includes; registration forms, login forms, contact us forms etc.

Forms are created in HTML using the <form> tag. It is a container for a form. The <form> tag is followed by the <input>, <textarea>, <select> tags and other form elements.
