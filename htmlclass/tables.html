HTML Tables

HTML tables are used to organize data into rows and columns. They can be used for displaying tabular information such as a list of products with their prices or a table showing the results of an examination. We also use tables when designing a dashboard

In this lesson we will learn how to create basic HTML tables using tags like <table>, <tr> (row), <td> (cell) and <th> (header).

We'll also cover some advanced topics including colspan attribute and rowspan attribute which allows us to span multiple cells across rows and columns respectively.

  • <table>: This element defines a table
  • <tr>:This defines a table row
  • <th>: Defines a header cell in the table
  • <td>: Defines a cell in the table, You can term it table data
